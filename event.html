pylekture.event¶

The Event Class An Event is always in a project and it may be in one or several scenario. Event is the baseclass for Scenario and Project. It inherits from Node, and add some attributes as wait, post_wait, loop and autoplay. It adds a few methods too as play(), getduration() and getparent()

class pylekture.event.Event(*args, **kwargs)¶

Bases: object

An Event is the base class for all events It has at least a parent. Only Projects does not have a parent. All objects refer to the projects these have created with. An optional name, description and tags attributes can be used. The service is a read-only value used to check the obect used. It should be removed for the version 0.1
